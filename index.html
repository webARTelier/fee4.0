Willkommmen bei FEE

FEE (Front End Experience) ist ein Framework, das Frontend Developern eine professionelle Grundlage für strukturiertes, konsistentes und skalierbares CSS bietet. Wichtige grundlegende Elemente jeder Website (Layout, UI, Behaviour) sind bereits vorgefertigt und können mit wenigen Handgriffen jederzeit an die eigenen Bedürfnisse angepasst werden – auch während oder sogar nach Fertigstellung des Projekts.

Für strukturierten, konsistenten und skalierbaren Code setzt FEE konsequent auf

  • namespaced BEM
  • CSS Custom Properties („CSS-Variablen“)1
  • CSS-Preprozessor SASS (SCSS)
  • Layout-Elemente mit CSS-Grid1
